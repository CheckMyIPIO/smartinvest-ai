# SmartInvest AI

A minimal, clean **Next.js 14 + Tailwind CSS** starter for SmartInvest AI — ready for Vercel.

## Quick Start

```bash
pnpm i # or npm install / yarn
pnpm dev # or npm run dev
```

## Deploy to Vercel
1. Create a new Vercel project and import this repo.
2. Framework preset: **Next.js**.
3. Build command: `next build` — Output: `.next` (default).
4. Add environment variables as needed (e.g., `NEXT_PUBLIC_OPENAI_KEY`).
